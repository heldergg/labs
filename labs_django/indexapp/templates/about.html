Acerca

Este site serve para testar ideias que não encaixam nos outros sites que mantemos e que, ao mesmo tempo, poderão ter alguma utilidade para os cidadãos.

Começamos com as seguintes aplicações: uma aplicação para consultar os resultados das eleições legislativas e para simular a composição do parlamento caso o número de deputados seja alterado; uma outra aplicação para calcular câmbios baseada nas taxas de câmbio de referência publicadas diariamente pelo BdP e, finalmente, uma aplicação para calcular a desvalorização da moeda.

Qualquer comentário, crítica ou sugestão pode ser enviado para: webmaster@tretas.org.

Nota técnica

Este sistema corre sobre Linux, utilizam-se os seguintes sistemas principais:

  • Base de dados PostgreSQL;
  • Interface web feita com Django;

Corre num servidor HP ML310 G8.

Desenvolvimento deste sistema

Todo o site foi feito usando software livre.

A escolha óbvia para a licença do próprio sistema foi a GPL v3. Pode obter o código fonte do site, colaborar no desenvolvimento, abrir bug reports, etc, na página do projecto no github.
