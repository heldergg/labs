Notas

Nesta aplicação são mostradas as taxas de câmbio publicadas diariamente pelo Banco de Portugal, disponíveis nesta página.

O BdP mantém um ficheiro com câmbios desde 1999-01-04 para todas as moedas cotadas diariamente que usamos para actualizar a nossa base de dados.

Estas taxas de câmbio de referência são calculadas diariamente com base no procedimento de concertação diária entre os bancos centrais dentro e fora do Sistema Europeu de Bancos Centrais, que ocorre por volta das 14:15 CET (tempo central europeu, 13:15 hora de Lisboa).

Pode haver dias em que a cotação de uma dada moeda não está disponível, quando for esse o caso o sistema tenta encontrar a cotação de data anterior à escolhida. Sempre que isso acontece é mostrado um aviso.
